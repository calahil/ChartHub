# Install on Android

The ChartHub Android companion app is sideloaded from the APK released on GitHub.

---

## Requirements

- Android 10 or later
- USB cable or network access to your desktop running ChartHub
- "Install from unknown sources" enabled for your browser or file manager

---

## Enable Unknown Sources

The steps vary by manufacturer, but in general:

1. Open **Settings → Security** (or **Settings → Apps → Special app access**).
2. Enable **Install unknown apps** for the app you will use to open the APK (e.g. your browser or a file manager).

---

## Download and Install

1. On your Android device, open the [Releases](https://github.com/calahilstudios/charthub/releases/latest) page in a browser.
2. Download `ChartHub-android.apk`.
3. Tap the downloaded file and follow the install prompt.

---

## Pair with Desktop

After installing, open the app and follow the [Connect to ChartHub Server](connect-to-server.md) guide to sign in.

---

## Updating

Download and install a newer APK — Android will upgrade the existing installation automatically.
