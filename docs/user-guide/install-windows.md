# Install on Windows

## Requirements

- Windows 10 or Windows 11
- .NET 10 Desktop Runtime (if not using the self-contained build)

---

## Download

1. Go to the [Releases](https://github.com/calahilstudios/charthub/releases/latest) page.
2. Download `ChartHub-win-x64.zip` (self-contained) or the installer if available.
3. Extract the archive to a folder of your choice.

---

## First Run

1. Open `ChartHub.exe` from the extracted folder.
2. On first launch ChartHub will create its configuration directory at `%APPDATA%\ChartHub`.
3. Log in with your RhythmVerse credentials when prompted.

---

## Optional: Add to Start Menu or Taskbar

Right-click `ChartHub.exe` and choose **Pin to Start** or **Pin to taskbar**.

---

## Updating

Download the latest release and replace the contents of the install folder. Configuration in `%APPDATA%\ChartHub` is preserved.
